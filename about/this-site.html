Overview

This website synthesizes a number of powerful web technologies to both allow me to curate it easily and to give visitors the best experience. Here's a list of what I use:

  • Foundation, for a CSS framework;
  • Jekyll, for blogging and templating;
  • GitHub, for hosting and version control;
  • Disqus, for blog comments;
  • Libertine Open Fonts Projekt, for sans and sans-serif type;
  • Liberation Fonts, for monospace type;
  • MathJax for $\LaTeX$ math typesetting;
  • Font Awesome, for symbol and glyph typesetting;
  • google-code-prettify, for code syntax highlighting;
  • jQuery, for easier Javascript;
  • Google Analytics, for audience analytics; and
  • Pandoc, for converting various text file formats.
